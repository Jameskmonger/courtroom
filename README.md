## Contribute

### Setup :wrench:
1. Clone this repository:
    `git clone https://github.com/Jameskmonger/courtroom.git`
2. Install the NPM modules:
    `npm install`
3. Compile the TypeScript - they will automatically be placed into the /build/ folder:
    `gulp`
4. Run the tests:
    `gulp test`
